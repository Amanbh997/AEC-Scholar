Draft a submission cover letter for: **$ARGUMENTS**

Use the **academic-writer** agent and `academic-writing` + `aec-journals`.

Write a concise, professional letter (typically ≤1 page) addressed to the Editor-in-Chief that:

1. **Opens** with the manuscript title, type (research article / review / etc.), and the journal it's
   submitted to.

2. **States the contribution** in 2–4 sentences: the problem, the gap, what the paper does, and the key
   finding — framed for **why it fits this journal's scope and readership** (use `aec-journals` scope notes).

3. **Makes the significance explicit** — scholarly and practical/industry relevance (AEC editors value both).

4. **Includes the standard declarations:** the work is original, not under consideration elsewhere, all
   authors approve submission, and any conflicts of interest / funding / prior-related-work (e.g. a
   conference version that this extends) are disclosed. Add a data-availability note if relevant.

5. **Optionally** suggests suitable (non-conflicted) reviewers or notes excluded reviewers, if the journal
   invites this.

6. **Closes** courteously with corresponding-author contact details.

Keep it specific to the actual manuscript and journal — no generic boilerplate, no over-claiming. Mark
placeholders (author names, grant numbers, prior version DOI) for the user to fill. Don't fabricate any
declaration; if you don't know whether a conference version exists, ask.
